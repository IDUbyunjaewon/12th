그밑의 스터디 준비,책구매,철수와 약속을 클릭하면 다음 화면으로 넘어가집니다 (스터디준비하기,책구매,철수와약속.png 3개의 파일 참고) 각각의 3개를 클릭후 MainView를 누르면 처음의 화면처럼
